3D Model Loader Tablet
--------------------

Load 3D models in-game using a tablet !

This VRChat World gimmick allows you to load 3D
models while inside a VRChat world, by just providing
the URL of the 3D model.

As it uses the 3D Model Loader as a backend, this
only supports GLB models (and a limited subset of VRM 1.0).
